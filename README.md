# AGROTECH 🌱

AGROTECH is a smart agriculture monitoring and analytics dashboard designed to provide real-time insights into agricultural processes through interactive dashboards and advanced data visualization.

## Features
- 🏠 Interactive Home Dashboard
- 📊 Stage Analysis Monitoring
- 📈 Advanced Analytics with IBM Cognos Integration
- 📜 History Log Tracking
- 🔐 Secure Login / Logout System
- 🎨 Modern Green-Themed Responsive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript
- IBM Cognos Analytics
- LocalStorage Authentication

## Purpose
This project helps monitor agricultural data efficiently by providing a centralized dashboard for analysis, visualization, and historical tracking.
